# 🥔 감자봇

## 📌 프로젝트 개요

슬랙봇을 생성하고 서버를 구축하여 직접 배포해봅니다.

## 🛠️ 기술 스택

<img src="https://skillicons.dev/icons?i=python,git,github"/>

### Python, Slack, NCP(NAVER CLOUD PLATFORM)

## 주요 기능

### 🤖 키워드 기반 자동응답 시스템 구현
|키워드 |응답     |
|-----|--------|
|감자야 |안녕, `사용자명`! 나는 찜감자야. 수줍..|
|심심해 |`사용자명`, 심심하구나. 감자농장에서 사람들과 이야기를 나눠보는 거 어때?|

### 📋 키워드 목록 조회 구현
'/감자어목록' 입력 시 키워드 목록을 조회하는 기능

### 🎉 이모티콘 추가 시 알림 기능
슬랙에서 사용자가 새 이모티콘을 추가하면, 봇이 이를 감지하고 알림을 보내는 기능 구현
> 감자농장에 `이모티콘명`이/가 이사왔습니다.

### 사용자가 채널 참여 시 봇 인사말 보내기
슬랙에서 사용자가 `#소식감자` 채널에 참여하면 봇이 인사를 건넵니다.
> 안녕, `사용자명`! 나는 소식해서 시무룩한 찜감자봇이라고 해

### 봇 메시지 전체 삭제 기능 구현
'/클린봇' 입력 시 봇 메신저 전체 삭제하는 기능
